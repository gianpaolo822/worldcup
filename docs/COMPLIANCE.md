# 知识产权与合规（摘要）

本项目为**个人学习**、**非官方**数据查询工具。

## 禁止

- FIFA / 世界杯官方 Logo、吉祥物、官方海报
- 各国足协官方队徽、球员官方照片
- 直播、集锦、新闻全文转载
- 文案中的「官方」「FIFA 授权」等表述

## 允许

- 赛程、比分、积分、球员姓名与统计等**文本/数字**事实数据
- 开放许可数据源（如 openfootball），并记录在 `data/DATA_SOURCES.md`

## 免责声明（用户端 Web 页脚 / 「关于」可参考）

```text
本站为个人学习项目，与 FIFA、国际足联、各国足协及任何官方机构无关。
所展示赛程、比分、积分等仅供参考，可能存在延迟或误差，请以官方发布为准。
```

完整风险表见 Cursor 计划文档「知识产权与侵权风险规避」章节。
