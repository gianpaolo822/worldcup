# 2026年FIFA世界杯竞赛规则

> **赛事全称**：2026年国际足联世界杯（FIFA World Cup 26）  
> **举办时间**：2026年6月11日 - 7月19日（共39天）  
> **举办地点**：美国、加拿大、墨西哥（首次三国联办）  
> **参赛球队**：48支（史上首次扩军）  
> **总场次**：104场  
> **官方口号**：We are 26  
> **规程版本**：FIFA理事会2025年5月9日批准 / 2026年3月19日修订版

---

## 一、赛制概览

### 1.1 小组赛阶段

| 项目 | 详情 |
|------|------|
| 小组数量 | 12个小组（A组至L组） |
| 每组球队 | 4支 |
| 小组赛场次 | 72场（每组6场 × 12组） |
| 晋级球队 | 32支（每组前2名 + 8个成绩最好的第3名） |

**小组赛分组原则**：
- 48支球队分为4档，每档12支球队
- 每组由4档各抽取1支球队组成
- 同一洲际协会的球队（欧洲除外）通常不会被分入同一小组
- 欧洲球队因晋级数量最多，允许同组存在

### 1.2 淘汰赛阶段

| 轮次 | 参赛球队 | 时间 |
|------|----------|------|
| 1/16决赛（三十二强） | 32支 | 6月28日-7月3日 |
| 1/8决赛（十六强） | 16支 | 7月4日-7月7日 |
| 1/4决赛（八强） | 8支 | 7月9日-7月11日 |
| 半决赛 | 4支 | 7月14日-7月15日 |
| 三四名决赛 | 2支 | 7月18日 |
| **决赛** | **2支** | **7月19日** |

- 淘汰赛共32场比赛
- **决赛场地**：美国新泽西大都会人寿体育场

### 1.3 总场次与时间安排

- **总场次**：104场（小组赛72场 + 淘汰赛32场）
- 相比2022年卡塔尔世界杯增加40场比赛
- 每支球队至少有3场小组赛，最多可踢8场（进入决赛）
- 球队首场比赛与次场比赛之间至少有**3天休息期**

## 二、积分规则

### 2.1 基本积分

| 结果 | 积分 |
|------|------|
| 胜 | 3分 |
| 平 | 1分 |
| 负 | 0分 |

### 2.2 小组排名规则

当两支或多支球队积分相同时，按以下**优先级**依次比较：

1. **积分** — 总积分高者排名靠前
2. **总净胜球** — 所有小组赛净胜球多者靠前
3. **总进球数** — 所有小组赛总进球多者靠前
4. **相互比赛积分** — 涉及球队之间比赛的积分
5. **相互比赛净胜球** — 涉及球队之间比赛的净胜球
6. **相互比赛进球数** — 涉及球队之间比赛的总进球数
7. **公平竞赛积分** — 红黄牌扣分较少者靠前（详见第七节）
8. **FIFA世界排名** — 赛前FIFA排名高者靠前
9. **抽签** — 以上全部相同时由FIFA随机抽签决定

> **注**：对于三队或以上同分的情况，先构建"相互比赛mini-table"进行子排名，再回归通用标准。

### 2.3 小组第三晋级规则

12个小组的第三名中，取成绩最好的**8支**晋级32强。跨组比较标准：

1. **积分**（优先级最高）
2. **净胜球**
3. **进球数**
4. **公平竞赛积分**
5. **FIFA世界排名**
6. **抽签**

**出线分数参考**（借鉴历届大赛经验）：
- **4分及以上**：基本确定晋级
- **3分**：通常足够，但不保证，净胜球至关重要
- **2分及以下**：出线希望渺茫

## 三、淘汰赛规则

### 3.1 基本规则

- **单场淘汰制**：90分钟内胜者晋级
- 淘汰赛**不存在客场进球规则**（世界杯为赛会制，无主客场概念）

### 3.2 平局处理方式

若90分钟常规赛时间结束比分持平：

1. **加时赛**：进行30分钟加时（上下半场各15分钟），中场不休息（球员留在场上）
2. **点球大战**：加时赛仍平局则进行点球决胜
   - 每队先踢5轮，5轮后仍未分出胜负则进入"突然死亡"（sudden death）
   - 罚球队员**不可与加时赛被换下场的球员重复**（即加时赛被换下者不得参加点球）

### 3.3 点球大战程序

- 裁判通过抛硬币决定球门和罚球顺序
- 每轮双方各派1名不同球员罚球
- 仅场上11名球员（含门将）有资格罚球，不可使用已换下球员
- 罚球前裁判确认球员身份

## 四、2026年世界杯新规（2026年3月FIFA公布）

为提升比赛节奏、减少时间浪费，2026年世界杯实施以下**4项新规**：

### 4.1 换人限时（10秒规则）

- 被替换下场的球员必须在换人牌出示后**10秒内**离场
- 若未能在10秒内离场：
  - 该球员仍必须退场
  - 替补球员需等到比赛计时满**1分钟后**的首次死球方可入场
- 此规则旨在杜绝"拖延战术式换人"

### 4.2 界外球与球门球限时（5秒规则）

- 执行界外球和球门球时，发球队员必须在**5秒内**将球发出
- 若倒计时结束球仍未发出，**球权将判给对手**
- 旨在加快比赛节奏，减少发球队员故意拖延时间

### 4.3 场外治疗规则（1分钟离场）

- 若球员因伤在场上接受评估，或其伤势导致比赛中断
- 该球员须在比赛重新开始后**离场并在场边停留1分钟**
- 旨在减少因伤停导致的比赛中断时间，同时确保球员得到适当评估

### 4.4 VAR规程调整

VAR介入范围新增两项：

1. **第二张黄牌导致的红牌**：VAR可介入因明显错误的第二张黄牌判罚导致的红牌
2. **角球误判**：VAR可复核明显错误判给的角球，前提是复核可立即完成且不延迟比赛恢复

**其他重申规则**：
- 仅场上**队长**可与裁判进行交流
- 其他球员如围堵裁判将被出示**黄牌**

## 五、球员与报名规则

### 5.1 大名单规模

| 名单类型 | 人数 | 说明 |
|----------|------|------|
| 初选名单 | 最多55人 | 需在赛前提交 |
| 最终名单 | **26人** | 含3名门将 |

- 最终名单中必须包含至少**3名守门员**
- 伤病替换：在球队首场比赛开球前24小时内，可因严重伤病进行人员替换
- 替换球员必须来自初选55人名单

### 5.2 换人规则

- 每场比赛常规时间最多可换**5人**
- 加时赛可额外换**1人**（共6人）
- 换人次数：最多**3次**换人时机（不含中场休息）
- 被换下球员不得再次上场

### 5.3 球员释放（俱乐部→国家队）

- **强制释放期**：2026年5月25日开始
- 球员必须在俱乐部最后一场官方比赛后（2026年5月24日）释放
- **例外**：参加洲际俱乐部赛事决赛的球员，可延迟至2026年5月30日释放（需FIFA批准）

### 5.4 球员资格

- 必须符合FIFA球员身份规定（出生地、父母/祖父母出生地、居住满5年等）
- 已代表某协会参加正式比赛的球员不可更换协会

## 六、公平竞赛规则

### 6.1 公平竞赛积分

| 处罚类型 | 扣分 |
|----------|------|
| 黄牌 | -1分 |
| 两黄变一红（间接红牌） | -3分 |
| 直接红牌 | -4分 |
| 黄牌+直接红牌 | -5分 |

### 6.2 公平竞赛积分使用场景

- **小组赛同分决胜**：作为第7优先级
- **小组第三排名**：作为第4优先级
- 扣分较少（纪律更好）的球队排名靠前

### 6.3 公平竞赛奖

赛事结束后，FIFA将根据公平竞赛积分评选"公平竞赛奖"（FIFA Fair Play Trophy），授予纪律表现最出色的球队。

---

## 七、弃赛与不可抗力

### 7.1 球队弃权处理

根据FIFA规程第6条：

- **赛前弃权**：FIFA可决定取消该队所在小组全部赛事，或寻找替代球队
- **小组赛阶段弃权/除名**：该队所有已赛结果被**宣告无效**（null and void）
- FIFA保留全权处理权，可采取任何必要措施

### 7.2 不可抗力中断比赛

若比赛因不可抗力（如雷暴、极寒、停电、看台骚乱等）中断：

1. **24小时内续赛**：从中断的那一分钟开始继续比赛，保持原有比分
2. 续赛时：
   - 使用相同的场上球员和可用替补
   - 不可新增替补
   - 剩余换人次数不变
   - 已被罚下球员不可替换
3. **纪律处罚**：中断前已执行的纪律措施仍然有效

> **注**：世界杯原则上不设完整重赛，通过"续赛机制"解决比赛中断。

### 7.3 比赛中止判定

若因非竞技性原因（如看台骚乱）导致比赛无法进行：
- 纪律委员会可能直接判处某一方0-3告负
- 而非安排重赛

## 八、场地与训练营

### 8.1 训练场地

- 每支球队拥有**官方训练营基地**（Team Base Camp）
- 训练营在比赛开始前至少5天可用，至球队被淘汰后1天
- 训练场地需符合国际比赛场地尺寸标准
- 赛前1个月内不得在官方训练场地举办其他活动（FIFA书面批准除外）

### 8.2 球队抵达

- 每支球队须在首场比赛前**不晚于5天**抵达东道国
- 使用FIFA指定的官方酒店（Team Base Camp Hotels）
- 淘汰赛阶段（1/16决赛起），球队释放训练营基地，改为"场馆跳跃"模式

### 8.3 友谊赛/热身赛

- 球队可在抵达东道国至首场比赛前5天之间安排热身赛
- **禁止事项**：
  - 同一小组的球队之间不得进行热身赛
  - 不得使用世界杯决赛阶段球场进行热身赛
  - 开赛前5天起不得在官方训练场地进行商业性质的热身赛
- 热身赛须使用FIFA指定的中立裁判执法

## 十、其他重要规定

### 10.1 比赛用球

- 使用FIFA官方指定比赛用球
- 每场比赛提供备用球，由FIFA统一安排

### 10.2 裁判安排

- 所有裁判来自FIFA国际裁判名单
- 使用VAR（视频助理裁判）系统
- 半自动越位技术（SAOT）将投入使用

### 10.3 装备规定

- 球队装备须符合FIFA装备规定
- 球衣号码范围为1-26号
- 队长须佩戴袖标
- 仅队长可与裁判进行正式交流

### 10.4 药物与反兴奋剂

- 适用FIFA反兴奋剂条例
- 赛事期间将进行兴奋剂检测
- 球队须指定队医负责医疗事务

### 10.5 纪律处罚

- 累计两张黄牌自动停赛一场
- 红牌停赛场次由FIFA纪律委员会决定
- 黄牌累计在小组赛结束后清零（不延续至淘汰赛），但淘汰赛期间累计两张黄牌仍会导致停赛
