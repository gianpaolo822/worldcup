# 2026 世界杯观赛指南 · 项目状态说明

**更新日期**：2026-06-04  
**项目性质**：个人学习项目 · **非 FIFA / 非官方**  
**完整 PRD**：[`PRD-2026-worldcup-guide.md`](PRD-2026-worldcup-guide.md)

---

## 一句话

我们正在做一款**中文移动端 Web 观赛手册**：查赛程、看积分、点进比赛看阵容和进球换人——不用装 App、不用注册；数据由公开赛报整理，**不是实时直播比分**。

---

## 你现在能用到什么

| 能力 | 说明 |
|------|------|
| **赛程** | 默认看「当天」比赛，可按日期区间、按球队筛选 |
| **积分榜** | 12 个小组 + 2026 扩军淘汰赛对阵图 |
| **数据榜** | 进球、助攻、黄红牌（来自已录入的比赛事件） |
| **历史** | 历届冠军、历史射手 |
| **比赛详情** | 阵型图、替补、换人/进球/黄牌时间线、技术统计 |
| **球队 / 球员** | 名单、中文名（持续补充）、伤退说明、队内数据 |

**本地预览**（维护者）：`worldcup-web` 目录运行 `npm run dev:clean`，浏览器打开 `http://localhost:5173/`。

**公开链接**：部署完成后将在此文档或 README 中更新。

## 数据覆盖（诚实说明）

| 指标 | 现状 |
|------|------|
| 赛程总场次 | 104 场（48 队） |
| 已更新比分/赛果 | 46 场 |
| 已录入阵容与事件 | 46 场（约 44%） |
| 球员中文名 | 39 支国家队 locale 文件，**仍在补充** |

未录入详情的比赛，详情页会显示占位提示；**比分与积分以已维护数据为准**，可能与电视直播有几分钟到数小时延迟。

---

## 明确不做什么（一期）

- ❌ 官方直播、视频、队徽 Logo  
- ❌ 实时推送、账号登录、评论社区  
- ❌ 微信小程序（验证通过后再评估）

页脚有**非官方免责声明**；全站仅用 emoji 国旗，避免侵权素材。

1. **上线** — 部署到公网，微信内自测

## 我们需要你的反馈（试用时请关注）

1. 赛程/积分/详情是否**够用**？还缺什么？
